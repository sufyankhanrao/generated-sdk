
# Fota V1 Success Result

A response to a successful request contains a single Boolean value.

## Structure

`FotaV1SuccessResult`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `success` | `bool` | Optional | True is returned in case of success. |

## Example (as JSON)

```json
{
  "success": true
}
```
