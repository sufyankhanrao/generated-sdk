
# Add Terminals Enum

The addTerminal field indicates whether or not all the terminals that exist on the template mid will be created for the submerchant. This is an optional field. If it is omitted, the value will default to “Yes”. If the value is “Yes”, all terminals on the template mid will be created for the submerchant. If “No”, no terminals will be created for the submerchant using this call. Terminals can be created using the Create Terminal endpoint.
