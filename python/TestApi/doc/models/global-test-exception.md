
# Global Test Exception

To test specific global exceptions.

## Structure

`GlobalTestException`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `server_message` | `str` | Required | Represents the server's exception message |
| `server_code` | `int` | Required | Represents the server's error code |

## Example (as JSON)

```json
{
  "ServerMessage": "ServerMessage8",
  "ServerCode": 212
}
```
