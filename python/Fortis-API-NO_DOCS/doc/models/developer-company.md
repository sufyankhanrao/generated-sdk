
# Developer Company

Developer Company Information on `expand`

## Structure

`DeveloperCompany`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `title` | `str` | Optional | Title<br><br>**Constraints**: *Maximum Length*: `64` |
