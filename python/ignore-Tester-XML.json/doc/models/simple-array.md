
# Simple Array

A model containing a string array with no xml attributes. This model's xml name should be "useThis"

## Structure

`SimpleArray`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `elem` | `List[str]` | Required | - |

## Example (as XML)

```xml
<useThis>
  <elem>elem3</elem>
  <elem>elem2</elem>
</useThis>
```
