
# Reallocation Freq 5

Describes the frequency of re-allocation. If set to 'yearly',             GOE would assume the re-allocation to happen once a year.

*This model accepts additional fields of type Any.*

## Enumeration

`ReallocationFreq5`

## Fields

| Name |
|  --- |
| `YEARLY` |
| `HALFYEARLY` |
| `QUARTERLY` |

## Example

```
yearly
```
