
# Risk Profile

Defines the user’s risk profile – does not vary by goal for each investor. <br>         On a default basis, GOE is configured for three Risk Profile levels, but it can be customized for up to five levels.

*This model accepts additional fields of type Any.*

## Enumeration

`RiskProfile`
