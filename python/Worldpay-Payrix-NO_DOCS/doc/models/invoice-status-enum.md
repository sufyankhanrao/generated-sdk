
# Invoice Status Enum

## Enumeration

`InvoiceStatusEnum`

## Fields

| Name | Description |
|  --- | --- |
| `PENDING` | Initial status of invoice before any action is taken. |
| `CANCELLED` | Updated status to indicate the invoice has expired. |
| `EXPIRED` | Updated status once the invoice has expired. |
| `VIEWED` | Updated status once the invoice is viewed by the customer. |
| `PAID` | Updated status once the invoice has been paid by the customer. |
| `CONFIRMED` | Updated status once the invoice has been confirmed by the customer. |
| `REFUNDED` | Updated status once the invoice has been refunded. |
| `REJECTED` | Updated status once the invoice has been rejected by the customer. |
