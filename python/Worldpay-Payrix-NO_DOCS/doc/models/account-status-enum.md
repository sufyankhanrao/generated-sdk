
# Account Status Enum

## Enumeration

`AccountStatusEnum`

## Fields

| Name | Description |
|  --- | --- |
| `NOTREADY` | Not Ready. The account holder is not yet ready to verify the Account. |
| `READY` | Ready. The account is ready to be verified. |
| `CHALLENGED` | Challenged. The account has processed the challenge. |
| `VERIFIED` | Verified. The Account has been verified. |
| `MANUAL` | Manual. There has been an issue during verification and further attempts to verify the Account will require manual intervention. |
