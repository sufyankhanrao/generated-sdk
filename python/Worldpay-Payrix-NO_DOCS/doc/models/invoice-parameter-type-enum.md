
# Invoice Parameter Type Enum

## Enumeration

`InvoiceParameterTypeEnum`

## Fields

| Name | Description |
|  --- | --- |
| `APIKEY` | The apikey that will be used for the invoice. |
| `FROMEMAIL` | The email the invoice will be sent from. |
| `REPLYTOEMAIL` | The reply-to email address for the invoice. |
| `DBA` | Name of the company that will be shown on the invoice. |
| `ADDRESS1` | Address1 shown on the invoice. |
| `ADDRESS2` | Address2 shown on the invoice. |
| `CITY` | City shown on the invoice. |
| `STATE` | State shown on the invoice. |
| `ZIP` | Zip code shown on the invoice. |
| `COUNTRY` | Country shown on the invoice. |
| `PHONE` | Phone number shown on the invoice. |
| `WEBSITE` | Website address shown on the invoice. |
| `RECEIPTTITLE` | The title when the invoice is sent. |
| `RECEIPTMESSAGE` | The message when the invoice is sent. |
| `BILLINGREQUIRED` | Whether the billing address will be required from the customer. |
| `SHIPPINGREQUIRED` | Whether the shipping address will be required from the customer. |
| `ECHECKAUTHORIZATION` | Authorization for ECheck payments. |
