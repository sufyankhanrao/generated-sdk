
# Skip Off Days Enum

## Enumeration

`SkipOffDaysEnum`

## Fields

| Name | Description |
|  --- | --- |
| `DONOTSKIP` | Do not skip Holidays and Weekends, Disbursement will be generated in a Requested status and process the next business day. |
| `SKIP` | Skip Holidays and Weekends, We do not advise setting this for weekly, monthly, or yearly Payout schedules as the disbursement will skip and not be generated until the next scheduled date. |
