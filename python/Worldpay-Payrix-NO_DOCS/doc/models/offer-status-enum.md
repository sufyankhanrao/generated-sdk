
# Offer Status Enum

## Enumeration

`OfferStatusEnum`

## Fields

| Name | Description |
|  --- | --- |
| `ACTIVE` | Indicates that the cash advance application is active and the sub-merchant is currently repaying the cash advance. |
| `APPLIED` | Indicates that the sub-merchant has applied for a cash advance and is awaiting approval from the vendor. |
| `APPROVED` | Indicates that the sub-merchant's cash advance application has been approved by the vendor and the sub-merchant is expected to start repaying the cash advance. |
| `EXPIRED` | Indicates that the cash advance application has expired and the sub-merchant is no longer eligible to receive the cash advance. |
