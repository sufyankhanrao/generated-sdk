
# Entity Custom Fields Post Request

## Structure

`EntityCustomFieldsPostRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `entity` | `str` | Optional | Entity for which we are creating custom field. |
| `key` | `str` | Optional | Name of custom field. |
| `value` | `str` | Optional | Value of custom field. |

## Example (as JSON)

```json
{
  "entity": "t1_ent_67d3ab7bc0ef1d4284d4578",
  "key": "companyId",
  "value": "70bbd35e-8500-f011-90cb-6045bdc7c168"
}
```
