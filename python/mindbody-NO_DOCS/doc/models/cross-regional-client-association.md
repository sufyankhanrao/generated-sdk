
# Cross Regional Client Association

A client cross region association

## Structure

`CrossRegionalClientAssociation`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `site_id` | `int` | Optional | The ID of the site to which the client belongs. |
| `client_id` | `str` | Optional | The client’s RSSID. |
| `unique_id` | `int` | Optional | The client’s unique ID. |
