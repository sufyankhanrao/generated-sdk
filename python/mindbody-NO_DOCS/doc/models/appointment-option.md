
# Appointment Option

An appointment option name/value pair

## Structure

`AppointmentOption`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `display_name` | `str` | Optional | The name displayed for this appointment option. |
| `name` | `str` | Optional | The name given to this option. |
| `value` | `str` | Optional | The value of the option. |
| `mtype` | `str` | Optional | The data type of the option value. |

## Example (as JSON)

```json
{
  "DisplayName": "DisplayName0",
  "Name": "Name6",
  "Value": "Value4",
  "Type": "Type6"
}
```
