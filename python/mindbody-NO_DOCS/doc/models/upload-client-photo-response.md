
# Upload Client Photo Response

## Structure

`UploadClientPhotoResponse`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `client_id` | `str` | Optional | The RSSID of the client for whom the photo was uploaded. |
| `photo_url` | `str` | Optional | The URL of the uploaded photo. |

## Example (as JSON)

```json
{
  "ClientId": "ClientId4",
  "PhotoUrl": "PhotoUrl8"
}
```
