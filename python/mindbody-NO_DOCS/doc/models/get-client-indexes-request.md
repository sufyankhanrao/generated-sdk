
# Get Client Indexes Request

## Structure

`GetClientIndexesRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `required_only` | `bool` | Optional | When `true`, filters the results to only indexes that are required on creation.<br /><br>When `false` or omitted, returns all of the client indexes. |

## Example (as JSON)

```json
{
  "RequiredOnly": false
}
```
