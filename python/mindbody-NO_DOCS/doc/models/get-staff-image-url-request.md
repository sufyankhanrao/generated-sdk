
# Get Staff Image URL Request

This is the request class for the get staff image URL API

## Structure

`GetStaffImageURLRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `staff_id` | `int` | Optional | The ID of the staff member whose image URL details you want to retrieve. |
