
# Get Mobile Providers Request

Get Mobile Providers Request Object

## Structure

`GetMobileProvidersRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `active` | `bool` | Optional | When `true`, the response only contains mobile providers which are activated.<br>When `false`, only deactivated mobile providers are returned.<br>Default: **All Mobile Providers** |

## Example (as JSON)

```json
{
  "Active": false
}
```
