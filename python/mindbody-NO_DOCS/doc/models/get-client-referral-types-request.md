
# Get Client Referral Types Request

## Structure

`GetClientReferralTypesRequest`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `include_inactive` | `bool` | Optional | When `true`, filters the results to include subtypes and inactive referral types.<br /><br>When `false`, includes no subtypes and only active types.<br>Default:**false** |

## Example (as JSON)

```json
{
  "IncludeInactive": false
}
```
