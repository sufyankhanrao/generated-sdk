
# Update Contact Log Comment

## Structure

`UpdateContactLogComment`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `id` | `int` | Optional | The ID of the contact log comment to be updated. Pass a value of `0` to add a new comment to the contact log. |
| `text` | `str` | Optional | The new text for the comment. |

## Example (as JSON)

```json
{
  "Id": 184,
  "Text": "Text8"
}
```
