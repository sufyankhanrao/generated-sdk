
# Power Curve Value

The power at a specific wind speed according to a power curve.

## Structure

`PowerCurveValue`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `wind_speed` | `float` | Required | Wind speed in m/s |
| `power` | `float` | Required | Power in kW |

## Example (as JSON)

```json
{
  "windSpeed": 64.16,
  "power": 27.74
}
```
