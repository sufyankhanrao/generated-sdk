
# Status Category Enum

The category a status belongs to.

## Enumeration

`StatusCategoryEnum`

## Fields

| Name | Description |
|  --- | --- |
| `STOP` | Statuses indicating that the device is prevented from producing power or is in a critical state. |
| `WARNING` | Statuses indicating that the device is not operating normally, but is not in a critical state. |
| `INFORMATIONAL` | Statuses that occur during normal operation. |
| `COMMUNICATION` | Statuses related to communication between the device and Greenbyte. |
| `CURTAILMENT` | Statuses generated based on curtailment. |
