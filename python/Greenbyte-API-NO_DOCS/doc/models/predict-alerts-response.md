
# Predict Alerts Response

An alert generated by Predict. The statusData object will be of one of the following types depending on its status:

* Predict Status Active
* Predict Status Resolved
* Predict Status Dismissed

## Structure

`PredictAlertsResponse`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `id` | `Any` | Optional | The id of the predict alert. |
| `device_id` | `int` | Optional | The id of the device the alert was generated for.<br><br>**Constraints**: `>= 1` |
| `site_id` | `int` | Optional | The id of the site the device is located in.<br><br>**Constraints**: `>= 1` |
| `component_alert` | [`Component`](../../doc/models/component.md) | Optional | A component of a wind turbine. |
| `high_severity` | `bool` | Optional | Whether the alert is high severity or not. |
| `status` | [`PredictStatusEnum`](../../doc/models/predict-status-enum.md) | Optional | The status of a Predict alert. |
| `status_data` | `Any` | Optional | - |
| `comments` | [`List[PredictComment]`](../../doc/models/predict-comment.md) | Optional | User comments for the alert. |
