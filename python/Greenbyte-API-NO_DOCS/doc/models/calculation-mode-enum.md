
# Calculation Mode Enum

Which operation to use when aggregating data.

## Enumeration

`CalculationModeEnum`

## Fields

| Name | Description |
|  --- | --- |
| `AVERAGE` | Returns the average of all data points in the range. |
| `SUM` | Returns the sum of all data points in the range. |
| `COUNTER` | Returns the raw accumulated counter data. |

## Example

```
sum
```
